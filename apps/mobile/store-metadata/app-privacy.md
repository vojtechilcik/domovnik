# App Privacy Details — Domovník (App Store + Google Play)

## Data Linked to User Identity

| Data Type | Purpose | Linked to Identity? |
|---|---|---|
| Email address | Authentication, communication | Yes |
| Name | Profile, lease agreements | Yes |
| Phone number | Contact for repairs | Optional |
| Address (property) | Property management | Yes |
| Payment history | Rent ledger | Yes |
| Photos (fault reports) | Repair assessment | Yes |
| Messages | Communication | Yes |

## Data Collection

- **Photos:** Only when a tenant reports a fault and chooses to attach a photo. Stored in EU-based object storage.
- **Photos are not accessed from the device gallery without explicit user action** (camera/gallery picker triggered by user).
- **Location data:** Not collected.
- **Device identifiers:** Not collected.

## Data Use

- All data is used solely for property management functions (lease management, rent tracking, repair handling, communication).
- Data is **not used for advertising, marketing, or tracking**.
- Data is **not shared with data brokers**.

## Account Deletion

Users can request account deletion at any time through:
- In-app Settings → "Smazání účtu a dat"
- Email: gdpr@domovnik.app

Deletion removes all user data permanently within 30 days.
